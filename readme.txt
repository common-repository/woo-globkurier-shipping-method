== Description ==

Globkurier Shipping Plugin for WooCommerce. Allows shipment via the service Globkurier.
WooCommerce is a powerful, extendable eCommerce plugin that helps you sell anything. Beautifully.

== Installation ==

= Minimum Requirements =

* WordPress 4.1 or greater
* WooCommerce 2.6 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you donâ€™t need to leave your web browser.
To do an automatic install of Globkurier Shipping, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New ...

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
